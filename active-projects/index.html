Linkmon

Open-source custom link generator and link manager. Alternative for bit.ly and linktree. Will have the following features.

  • Custom link generation
  • Link shortener.
  • Self hostable and can be added to websites easily.
  • Store and manage links as links trees.
  • Get analytics like number of clicks..etc

TinkerHub Logo Generator

This is a logo generator for TinkerHub Campus Chapters. TinkerHub Campus communities help students to learn and build tech with peer mentoring and self learning. These logos will be used in posters, videos and posts about the coolest initiatives student leaders are doing in their campuses, to spread technology learning. Contributing to this project makes you a part of that ❤️.

This project is complete. But there are bugs to fix and features to add.

Some of the issues and features are,

  • Issue with MacOS
  • Redesign the UI
  • Feature for others to generate their community chapter logos.

Standard Project template

This is a standard project template. You can fork this and use this template while creating a new project. GitHub will automatically add the files and configurations specified in the standard template repo to your new project.

Be creative and contribute to this repo to make the life of developers easy.

Triviator

Triviator is an open-source, self hosted platform for,

  • Ask questions and do polls during an online meeting.

TinkerHub-Bot

TinkerHub bot is a telegram bot for sharing resources based on the stack one like to learn and it’s levels

  • Share resources when asking for resources like say /resource machine learning.
  • General info/updates about tinkerhub and events we conduct.
  • Features for managing tinkerhub telegram group.
